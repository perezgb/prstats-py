# prstats-py

Get PR data using Python.

## getting started
Setup virtualenv:
```bash
virtualenv venv
```

Activate the virtualenv:

On a mac:
```bash
. venv/bin/activate
```

On windows:
```
venv/Scripts/activate.bat
```
Install the dependencies
```bash
pip install -r requirements.txt
```
